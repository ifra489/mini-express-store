# 🛒 Mini Express Store

A simple REST API built with Node.js and Express.js.

## 🚀 Getting Started

### Prerequisites
- Node.js installed

### Installation
```bash
git clone https://github.com/ifra489/mini-express-store.git
cd mini-express-store
npm install
```

### Run the server
```bash
node index.js
```

Server runs on `http://localhost:3000`

---

## 📌 API Routes

| Method | Route | Description |
|--------|-------|-------------|
| GET | `/` | Home - Welcome message |
| GET | `/products` | Get all products |
| GET | `/contact` | Contact info |
| GET | `/about` | About us |
| POST | `/register` | Register user |
